Step 1

Sign up for a free Auth0 account.

Step 2

In your Auth0 application settings:

Set {{url}}callback as an Allowed Callback URLs.

Set {{url.substring(0, url.length - 1)}} as an Allowed Logout URLs.

Set {{url.substring(0, url.length - 1)}} as an Allowed Web Origins.

Step 3

Within src/environments/environment.ts:

Replace the values of CLIENT_ID and CLIENT_DOMAIN with the values of Client ID and Domain from the Auth0 Application Settings.

Replace the value of REDIRECT with {{url}}callback

Replace the value of LOGOUT_URL with {{url.substring(0, url.length - 1)}}
